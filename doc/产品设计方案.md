# Angel Number Decoder — 完整产品设计方案

> **项目定位**: 以 Angel Number 解读为核心流量入口，结合 Gematria 计算与 AI 解密分析的数字神秘学工具平台
>
> **技术栈**: Next.js 16+ (App Router) + TypeScript + Tailwind CSS + MDX
>
> **目标**: 6 个月内达到 50K+ 月有机流量，建立 AI 解密工具的付费漏斗

---

## 1. 产品概述

### 1.1 核心价值主张

"输入任何数字，立即解锁它在宇宙中的隐藏含义。"

我们不只是一个计算器——我们是一个 **数字含义解读平台**，让用户从"看到一个数字"到"理解这个数字的深层意义"，最终引导到 AI 驱动的深度分析工具。

### 1.2 目标用户画像

| 用户类型 | 年龄段 | 特征 | 典型场景 |
|---------|--------|------|---------|
| 灵性探索者 | 18-35 | 关注 angel number、tarot、astrology | 看到 11:11 想查含义 |
| 命理爱好者 | 25-45 | 研究 numerology、name meaning | 给宝宝/公司取名 |
| 圣经/宗教研究者 | 30-60 | 研究 gematria、bible codes | 分析经文隐藏含义 |
| 解密/密码爱好者 | 20-40 | 喜欢 cipher、puzzle、ARG | 解码神秘文字 |

**核心组件:**

- **Hero 搜索框**: 页面居中大搜索框，placeholder: "Enter any number to decode its meaning..."，支持输入数字后直接跳转到 `/angel-number/[number]/`

### 3.3 Gematria Calculator (/calculator/gematria/)

**功能:**
- 输入框：接受英文/希伯来文
- 实时计算：打字即算，无需点击按钮
- 多系统同时显示：Hebrew Standard、English Ordinal、Simple (A=1)、Jewish、English Gematria (x6)、Full Reduction、Reverse Ordinal
- 逐字拆解：展示每个字母的对应数值
- 同值词匹配：显示数据库中同样数值的其他词/短语
- 分享功能：生成分享卡片 / 复制链接
- AI 解读 CTA："What does this number mean? → Get AI Interpretation"

**技术实现:**
- 所有计算纯前端 JavaScript，零 API 调用
- 使用 `useMemo` 缓存计算结果
- 字母映射表存储在 JSON 常量文件中

### 3.4 Numerology Calculator (/calculator/numerology/)

**功能:**
- 输入：Full Name + Birthday
- 计算并展示：
  - Life Path Number（生日数字求和递减）
  - Destiny/Expression Number（名字字母求和）
  - Soul Urge Number（元音字母求和）
  - Personality Number（辅音字母求和）
- 每个数字附带简要解读
- 可视化展示：数字轮盘 / 雷达图
- AI 深度解读 CTA

### 3.5 Programmatic SEO 页面生成策略

**需要覆盖的数字页面:**

| 类型 | 数字范围 | 页面数 | 示例 |

| 两位重复 | 11-99 | 9 | /angel-number/44/ |
| 镜像数字 | 1221, 1331... | ~20 | /angel-number/1221/ |
| 递增序列 | 123, 1234... | ~5 | /angel-number/1234/ |

**内容生成策略:**

- 其余页面：使用模板 + 数据驱动生成，但每个页面内容不同（基于数字的属性、组成数字的含义组合等）
- 每个页面包含独特的 FAQ、affirmation、biblical reference

## 4. UI/UX 设计规范

### 4.1 设计风格

**风格定位: Mystical Modern（神秘现代感）**

不走 Gematrix 那种 2010 年老旧风格，也不走那些灵性网站的"彩虹水晶"风格。我们要做的是 **暗色系 + 极简 + 微妙的神秘感**，类似 Linear.app 的设计品质 + 星空/数字的视觉元素。

参考风格：Cypher Numerology (numerology.pardesco.com) 的暗色调路线，但更加精致和现代。

type CipherSystem =
  | 'hebrew_standard'     // Mispar Hechrachi

  | 'full_reduction'      // Single digit reduction

interface GematriaResult {

  letterBreakdown: {
    letter: string;
    values: Record<CipherSystem, number>;

- **字体优化**: 使用 `next/font` 本地加载，避免 FOUT
- **图片**: 使用 `next/image` + WebP + 懒加载
- **JS Bundle**: 计算器组件 lazy load (`dynamic import`)
- **CSS**: Tailwind CSS purge 未使用样式
- **缓存**: Vercel Edge Cache + `s-maxage` 头
